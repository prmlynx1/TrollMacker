# 🚀 TrollMacker v1.0

### CMD Automation & Script Management Tool

TrollMacker is a developer utility designed to simplify **CMD automation, script management, and command generation** for Windows environments.

The tool helps users generate compact CMD scripts, manage multiple commands, and streamline automation tasks efficiently.

---

# 📜 Overview

TrollMacker focuses on improving productivity when working with **Windows Command Prompt automation**.  
It allows users to generate structured CMD commands, organize scripts, and execute tasks quickly.

---

# ⚙️ Features

🔹 Generate compact CMD command scripts  
🔹 Multi-command script generation  
🔹 Built-in script management system  
🔹 Secure access system  
🔹 Script protection & code obfuscation  
🔹 Build executable files from scripts  
🔹 Support for multiple file types  
🔹 Command automation for repetitive tasks  
🔹 Upload and manage multiple files per project  
🔹 Integrated database for managing generated CMD scripts  
🔹 Developer-friendly workflow tools
